This repository contains laboratory assignments completed as part of the SO1 (C Programming) course.

The labs are based on the *C Programming Workbook* by Marc Hensel and progressively build core programming skills in C.

---

📂 Labs Overview

 🔹 Lab 1 – Getting Started
- Basic C programs
- Input/output operations
- Integer calculations
- Digit sum implementation

 🔹 Lab 2 – Data Types & Control Flow
- Conditional statements and loops
- Chessboard field calculations
- Wheat and chessboard problem

 🔹 Lab 3 – Functions
- Modular programming with functions
- Geographic coordinate calculations
- Distance computations

 🔹 Lab 4 – Arrays & Pointers
- 1D and 2D arrays
- Pointer usage
- Route and distance calculations

 🔹 Lab 5 – Memory Management & Structures
- Dynamic memory allocation
- Structs in C
- Travel planning system

---

 🛠 Technologies
- Language: C
- IDE: Visual Studio
- Concepts: Algorithms, memory management, data structures

---

 📌 Notes
- Each lab is implemented as a separate project inside the same solution.
- Code follows standard C coding practices and structure.

---

 🚀 How to Run
1. Open the solution in Visual Studio
2. Select a lab project (e.g., Lab1)
3. Build and run
