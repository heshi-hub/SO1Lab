 Lab 1 – Getting Started

 📌 Description
This lab introduces the basics of C programming, including writing simple programs, handling input/output, and performing integer operations.

 🎯 Objectives
- Understand Visual Studio project structure
- Use `printf` and `scanf`
- Perform basic arithmetic operations
- Implement digit sum calculation

 🧠 Key Concepts
- Variables and data types
- Console input/output
- Basic algorithms

 ▶️ Example Tasks
- Print formatted output
- Read integers from user input
- Compute digit sum of a number

 🚀 How to Run
1. Open project in Visual Studio
2. Build and run
